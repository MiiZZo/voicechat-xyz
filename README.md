# VoiceChat

A Windows desktop voice/text chat application with screen sharing, built as an npm monorepo containing a Fastify lobby server (`@voicechat/server`) and an Electron + React client (`@voicechat/client`).

See the design spec at [docs/superpowers/specs/2026-04-26-voicechat-design.md](docs/superpowers/specs/2026-04-26-voicechat-design.md) for details.
